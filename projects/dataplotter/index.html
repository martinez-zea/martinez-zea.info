Data Plotter

2010

Data Plotter es un dispositivo que crea dibujos usando como input los datos capturados por el BereBere. Su principio es similar al de un sismógrafo. Su comportamiento esta determinado por la variación de los datos capturados durante las derivas. Esta maquina fue construida para realizar una acción que deja una huella sobre un papel.

Data Plotter is a device which create drawings from the data captured by the berebere. Its principle is similar to the seismograph. Its behavior is determined by the variation of the data captured during the wandering. This machine was built to perform an action which leaves traces on paper.
