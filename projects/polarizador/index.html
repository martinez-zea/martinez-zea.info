Polarizador

2008

Polarizador es una máquina que hace preguntas, almacena las respuestas y devuelve un recordatorio impreso de la interacción. Captura información y la compara para crear opiniones opuestas acerca de cualquier tema.

Cada visitante es identificado por el sistema por un código de barras. Cuando la máquina lee el código, selecciona una de un grupo de preguntas predefinidas y la lee por medio de un motor de texto a voz. El usuario responde presionando uno de dos botones ( si / no ), su respuesta es almacenada en una base de datos para realizar comparaciones y estadísticas posteriormente. El Polarizador imprime un recibo con la comparación de la respuesta actual con una almacenada en la base de datos y luego actualiza un simple infografía proyectada.

Polarizador is a machine that asks questions, stores the answers and returns printed reminders of the interaction. It captures and compares information to create opposite opinions about any subject.

Each visitor is identified by the system using a bidimensional barcode. When the machine reads a barcode it picks one from a limited set of predefined questions and reads it by means of a text to speech application. The user answers the question by pressing one of two buttons ( yes / no ) and his answers are stored in a database for further comparison and statistics. Polarizador prints a receipt that contains a comparison of the current answer with others stored in the database and updates a simple pie chart projected in the wall.
